# 🛡️ Microsoft Sentinel SOC Analyst Lab
## 📖 Overview

This repository documents my hands-on learning journey with **Microsoft Sentinel** and the **Microsoft Defender Portal** through Microsoft Learn and Microsoft Sentinel Ninja training.

The labs simulate real-world Security Operations Center (SOC) analyst tasks, including building detection rules, writing Kusto Query Language (KQL) queries, investigating incidents, and managing security alerts.

The goal of this project is to demonstrate practical experience with Microsoft Sentinel rather than only theoretical knowledge.
